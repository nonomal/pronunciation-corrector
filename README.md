## 功能
+ [x] 在线发音校正。
+ [x] 支持选择词库。
+ [x] 支持保存进度。
+ [ ] 显示音标以及翻译。
+ [ ] 跨设备设置与进度同步。
+ [ ] 支持收藏单词。
+ [ ] 支持自定义词库。
+ [ ] 支持分享词库。

## 已支持的词库
```json
{
  "ef-common-3000": "EF 常用 3000 词汇",
  "chinese-programmer-wrong-pronunciation": "中国程序员容易发音错误的单词",
  "computer-science-glossary": "计算机科学词汇表",
  "common-oral-2265": "英语口语中最常使用的 2265 个单词",
  "common-500": "最常用的 500 个英语单词"
}
```
